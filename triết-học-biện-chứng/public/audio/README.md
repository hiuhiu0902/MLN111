# Audio Files for Flipbook

Đặt các file audio MP3 của bạn vào thư mục này với tên như sau:

## Cấu trúc file audio:

- `page0.mp3` - Giọng đọc cho trang bìa
- `page1.mp3` - Giọng đọc cho trang 1 (/pages/2.png)
- `page2.mp3` - Giọng đọc cho trang 2 (/pages/3.png)
- `page3.mp3` - Giọng đọc cho trang 3 (/pages/4.png)
- `page4.mp3` - Giọng đọc cho trang cuối

## Hướng dẫn:

1. Chuẩn bị file MP3 với chất lượng tốt
2. Đặt tên file theo quy ước trên
3. Copy vào thư mục này
4. Nút 🔊 sẽ tự động hoạt động

## Lưu ý:

- Format: MP3 được khuyến khích
- Kích thước: Nên dưới 5MB mỗi file
- Chất lượng: 128kbps - 320kbps
